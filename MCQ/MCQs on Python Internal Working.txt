## MCQs on Python Internal Working

1. Python is mainly an ______ language.
   A) Compiled
   B) Interpreted
   C) Assembly
   D) Machine

   **Answer: Interpreted


---

2. Which program converts Python code into bytecode?
   A) Loader
   B) Compiler
   C) Interpreter
   D) Assembler

   **Answer: Compiler

---

3. Python source code is first converted into:
   A) Machine code
   B) Assembly code
   C) Bytecode
   D) Binary file

   **Answer: Bytecode 

---

4. Bytecode in Python is stored in:
   A) `.exe` files
   B) `.class` files
   C) `.pyc` files
   D) `.html` files

   **Answer: .pyc files

---

5. Which component executes Python bytecode?
   A) CPU
   B) PVM
   C) RAM
   D) BIOS

   **Answer: PVM

---

6. PVM stands for:
   A) Python Variable Machine
   B) Program Virtual Machine
   C) Python Virtual Machine
   D) Python Visual Machine

   **Answer: Python Virtual Machine

---

7. What is the correct flow of Python execution?
   A) Source Code → Machine Code → Bytecode
   B) Source Code → Bytecode → PVM
   C) Source Code → PVM → Compiler
   D) Bytecode → Source Code → PVM

   **Answer: Source Code → Bytecode → PVM



---

8. Which file contains Python source code?
   A) `.py`
   B) `.pyc`
   C) `.ppt`
   D) `.exe`

   **Answer: .py
---

9. Python is platform independent because:
   A) It uses machine code directly
   B) Bytecode can run on any system with PVM
   C) It uses only Windows
   D) It avoids compilation

   **Answer:  Bytecode can run on any system with PVM

---

10. Which of the following creates bytecode?
    A) Linker
    B) PVM
    C) Python Compiler

    **Answer: PVM

11. Python Virtual Machine is responsible for:
    A) Writing source code
    B) Executing bytecode

20. Which of the following correctly explains Python working?

    B) Source Code → Output → Compiler
